# Nightcraft

> A handcrafted dark theme by **Matt Wigg**

Nightcraft is a personal VS Code color theme built for long coding sessions in low light. It pairs near-black editor depths with measured steel-blue chrome accents, riding on an Atom One Dark syntax base for familiar, readable code colors.

---

## Features

- **Near-black editor** — `#121314` background, easy on the eyes in dark environments
- **Steel-blue accents** — active borders, badges, and selections use a calm `#3994BC` blue
- **Atom One Dark syntax** — battle-tested token colors for all major languages
- **Low-contrast chrome** — sidebar, activity bar, and title bar use a slightly lighter dark (`#191A1B`) to keep focus on the code
- **Semantic highlighting off** — clean, consistent token coloring without semantic overrides

## Installation

1. Open the **Extensions** view (`Ctrl+Shift+X`)
2. Search for `Nightcraft`
3. Click **Install**
4. Open the Command Palette (`Ctrl+Shift+P`) and select **Color Theme → Nightcraft**
